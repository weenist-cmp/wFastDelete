wFastDelete 간단설명

게임이나 프로그램들이 너무 많은 파일갯수와 용량때문에 삭제할 때 시간이 많이 걸려서 좀 빠른 프로그램을 찾아보니 마땅한게 없어서 직접 만들었습니다.

-구성
wFastDelete - 사용자 설명서.odt
wFastDelete - 간단 설명.txt
wFastDelete.exe
wFastDelete.ini
wFastDelete - InfoList (예제).txt

 
-설치, 사용법
아무폴더에 파일을 복사해두고 우클릭 관리자모드로 실행합니다.
[등록] 버튼을 클릭하면 이제부터 사용가능합니다.
등록한 이후에 해제하고 싶지 않다면 빈곳을 더블클릭해서 종료합니다.
설정을 바꾸고 싶을때도 다시 관리자모드로 실행해서 [등록] 버튼을 클릭합니다.

[자동닫기]는 폴더 삭제시나 폴더정보(LM)이 성공적으로 끝났을경우 창을 자동으로 닫아줍니다. 창을 닫는 시간은 wFastDelete.ini 파일을 메모장으로 열어 바꿀 수 있습니다.
[메세지 비표시]는 폴더 삭제시 삭제작업 결과를 보여주지 않습니다.
[LM]은 일반 사용자는 필요없지만 폴더안의 폴더와 파일들의 목록을 구할 때 사용하는 폴더정보LM)을 등록합니다. 이때 Directory / File / FileSize 를 선택합니다. 이때 선택한 항목이 기본으로 선택됩니다.

탐색기나 기타 프로그램에서 폴더에서 우클릭하면 나오는 메뉴중에
[폴더삭제]를 클릭하면 삭제가 실행됩니다. 삭제작업이 끝나면 결과가 출력됩니다.
[폴더정보]를 클릭하면 폴더의 정보를 보여줍니다.
[폴더정보 (LM)]을 선택하면 파일목록을 생성합니다. 목록종류를 선택하고 [ListMake]를 클릭하면 저장할 파일 위치와 이름을 지정하고 확인하면 폴더의 목록파일이 만들어집니다. 이때 이미 존재하는 파일이름이면 확인창이 표시됩니다.

-삭제법
관리자모드로 실행한후에 [해제] 버튼을 클릭하고 모든 파일을 삭제하면 됩니다.

-달라진점
이전버전에서 비표준 ANSI파일명이 지워지지 않을 경우가 있었지만 이젠 다른프로그램에서 사용하고 있지 않다면 모두 삭제가능합니다. 또한 다국어 언어를 지원했습니다. wFastDelete.ini를 메모장에서 열어보면 설명이 있습니다.
사용하는 글꼴은 다국어를 지원하려고 기획해서 기본 글꼴로 해서 그리 이쁘지 않습니다. 마찬가지로 창의 모양이 이쁘지 않은것도 간단하게 작은 용량으로 만들려고 해서 그렇고 메세지와 버튼의 간격이 넓은 이유는 폴더의 전체 길이를 넉넉하게 잡아서 그렇습니다.
우클릭 메뉴를 하위메뉴에 넣었습니다. 이때 아이콘이 표시되게 했습니다. 아이콘 변경방법이나 기타 여러가지 자세한 설정은 wFastDelete.ini 파일을 메모장에서 열어 설명을 잘 읽어보시고 주의해서 변경하시면 됩니다.
진행상황을 알려주는 막대바를 추가했습니다.
ShowWhileWorking 폴더정보를 볼때나 목록파일을 작성할때 진행상황 표시 (느림)
ShowWhileRemove 삭제작업할때 진행상황 표시 (느림)
CommaCount 숫자 중간에 쉼표 추가 (몇글자마다 0~10)
